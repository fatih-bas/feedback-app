## Feedback App

`Add, Delete and Edit Feedback`


- `React`
- `Context API`
- `Css`
- `React Router`



### Setup
1. Clone the repository and install the dependencies
```bash
Npm install
```
1. Start the application locally
```bash
Npm run dev
```
